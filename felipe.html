O curso de programação web tem como objetivo a capacitação de profissionais para desenvolvimento de sistemas web atuando em programação de front-end ou back-and. O curso tem carga horária de 240 horas e sua grade é composta por conteúdos de Front-end (HTML, CSS e JavaScript) e Back-end (PHP e SQL).


O senac conta com diversos cursos na área de programação com: Lógica de programação, programação de sistema, fullstack, formação em pythom.
